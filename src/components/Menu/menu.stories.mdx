# Menu
为网站提供导航功能的菜单。支持横向纵向两种模式，支持下拉菜单。
~~~js
// 这样引用
import { Menu } from 'tintui'
//然后可以使用 Menu.Item 和 Menu.Submenu 访问选项和子下拉菜单组件
~~~
