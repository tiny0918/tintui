# Select
下拉选择器。弹出一个下拉菜单给用户选择操作，用于代替原生的选择器，或者需要一个更优雅的多选器时。
### 引用方法
~~~js
import { Select } from 'tintui'
// 然后可以使用 <Select> 和 <Select.Option>
~~~
