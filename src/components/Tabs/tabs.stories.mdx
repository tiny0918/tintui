# Tabs
选项卡切换组件。
提供平级的区域将大块内容进行收纳和展现，保持界面整洁。
### 引用方法
~~~js
import { Tabs } from 'tintui'
~~~

## 默认样式的Tabs

<Preview >
  <Story name='defaultTabs' >

      <TabItem label="tab2">this is content two</TabItem>
      <TabItem label="tab3">this is content three</TabItem>
    </Tabs>
  </Story>
</Preview>

## 选项卡样式的Tabs

<Preview>
  <Story name='selectTab' >
