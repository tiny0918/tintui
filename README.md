## 安装

~~~javascript
npm install tintui --save
~~~

## 使用

~~~javascript
// 加载样式
import 'tintui/dist/index.css'
// 引入组件
import { Button } from 'tintui'
~~~

## 特性

* 提供完整的TypeScript类型定义文件
* 完整的单元测试
* 详细的文档与介绍
* 涉及全部流程,开箱即用

## 本地开发命令

~~~bash
//启动本地环境
npm run stroybook

//跑单元测试
npm test

//build可发布静态文件
npm run build

//发布到 npm
npm run publish
~~~
